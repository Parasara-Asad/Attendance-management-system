# Attendance Management System

A modern, responsive Attendance Management System built using **HTML**, **CSS**, and **JavaScript**.

## 🌐 Live Demo

🔗 https://parasara-asad.github.io/Attendance-management-system/

## 📂 Repository

🔗 https://github.com/Parasara-Asad/Attendance-management-system

## 🚀 Clone the Repository

```bash
git clone https://github.com/Parasara-Asad/Attendance-management-system.git
```

## 📁 Navigate to the Project

```bash
cd Attendance-management-system
```

## 💻 Run the Project

Simply open the `index.html` file in your browser.

Or, if you're using VS Code, install the **Live Server** extension and click **Go Live**.
