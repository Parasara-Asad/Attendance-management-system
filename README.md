# 📚 Attendance Management System

A simple and responsive **Attendance Management System** built using **HTML, CSS, JavaScript, and Bootstrap 5**. It allows users to generate roll numbers, mark attendance, and track present and absent students in real time.

---

## ✨ Features

- ✅ Enter Start Roll Number
- ✅ Enter End Roll Number
- ✅ Generate Attendance Numbers
- ✅ Navigate Using Previous & Next Buttons
- ✅ Mark Students as Present
- ✅ Mark Students as Absent
- ✅ Prevent Duplicate Attendance Entries
- ✅ Display Present Student List
- ✅ Display Absent Student List
- ✅ Display Total Present Students
- ✅ Display Total Absent Students
- ✅ Responsive and User-Friendly Interface

---

## 🛠️ Built With

- CSS3
- JavaScript (ES6)
- Bootstrap 5.3
