# Attendance Management System

An easy Attendance Management System built in **HTML, CSS, JavaScript, and Bootstrap**.

## 📌 Features

- Input Start Roll Number
- Input End Roll Number
- Generate list of students' attendance
- Mark students as Present
- Mark all students as Present
- Clear all attendance
- Display list of present students
- Display list of absent students
- Display total number of present students
- Display total number of absent students

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## 🚀 How to run

1. Download the project
2. Open `main.html` file
3. Use Attendance Management System
